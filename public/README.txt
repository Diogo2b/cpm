# Logo Corse Pose Menuiserie (CPM)

Inclui variantes prontas para integração em Astro.js:

- logo-cpm.svg                → vetor limpo (embed indireto)
- logo-cpm.webp               → imagem otimizada web (1024x1024)
- logo-cpm.png                → fallback universal (1024x1024)
- favicon-32x32.png           → ícone de navegador
- android-chrome-192x192.png  → ícone PWA
- android-chrome-512x512.png  → ícone PWA alta resolução

Colocar tudo na pasta /public/ do projeto Astro.

```astro
<img src="/logo-cpm.svg" alt="Logo CPM" width="180" height="auto" />
```
